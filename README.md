# Website Permission Analyzer

A Chrome Extension that visually displays all permissions used by the current website in a clean, simple dashboard.

## Features
- Displays camera, microphone, location, cookies, pop-ups, JavaScript, images, sound permissions
- Automatically detects the current website
- Clean UI with status indicators
- Built using Chrome Manifest V3

## Tech Stack
- HTML
- CSS
- JavaScript
- Chrome Extensions API

## How to Install
1. Clone this repository
2. Open Chrome and go to `chrome://extensions`
3. Enable **Developer Mode**
4. Click **Load unpacked**
5. Select the project folder

improving awareness and security.
